# Tool Radar — 2026-06

Window: 2026-05-08 → 2026-06-08
Audience: solo Unity dev, Windows + RTX GPU
Scoring: relevance to a solo Unity dev (1–10). Anything ≥7 is worth a serious look this month.

Legend: **★Upgrade** flags a direct upgrade path for one of the three target workflows: **CUI** = ComfyUI workflow, **B→U** = Blender-to-Unity pipeline, **EAI** = enemy AI systems.

---

## AI-Powered Tools

### 3D asset generation

**Tencent Hunyuan3D 3.5** — ★Upgrade (CUI)
- Open-source text/image-to-3D, now sub-60s generation with up to 8K PBR textures; 2× faster than 3.5 predecessor.
- Free (open-source weights, research-friendly license).
- Relevance: **9** — drop-in node upgrade to existing Hunyuan ComfyUI workflows; the PBR jump matters for game-ready output.
- RTX Windows: **yes** (community ComfyUI nodes run on consumer RTX, ~12GB VRAM for full quality).
- https://github.com/Tencent-Hunyuan/Hunyuan3D-2

**Microsoft Trellis 2** — ★Upgrade (CUI)
- 4B-param DiT pipeline, 1536³ voxel resolution, 4K PBR textures, better topology + transparent materials.
- Free (open-source, one-click ComfyUI Windows installer available via community).
- Relevance: **9** — currently the highest visual quality OSS 3D generator; pair with CHORD for PBR finishing.
- RTX Windows: **yes** (24GB recommended at full res; 12GB works at lower settings).
- https://trellis2.app/blog/best-ai-3d-model-generator

**Meshy-6**
- Hosted text/image-to-3D + auto-rig + PBR + engine export in one browser tool. Latest generation model in the Meshy line.
- Subscription: Free tier (200 credits/mo, watermark-free for personal), Pro $20/mo, Max $60/mo.
- Relevance: **8** — best-in-class hosted UX; only choice here that auto-rigs humanoids cleanly enough to drop straight into Unity.
- RTX Windows: **yes** (hosted — your GPU is irrelevant).
- https://www.meshy.ai/blog/best-ai-tools-for-3d-game-assets

### Character / animation AI

**Inworld Realtime TTS-2** (launched 2026-05-05)
- Closed-loop voice model with inline direction prompts (`[speak sadly]`, `[sigh]`, `[laugh]`) and conversational state across turns.
- Subscription / API: pay-as-you-go via Inworld API; pricing on platform.
- Relevance: **7** — best-in-class for AI NPC dialogue; only relevant if your game has live-generated speech.
- RTX Windows: **yes** (hosted API).
- https://www.marktechpost.com/2026/05/05/inworld-ai-launches-realtime-tts-2-a-closed-loop-voice-model-that-adapts-to-how-you-actually-talk/

**DeepMotion Real-Time Markerless Tracking** (2026 rollout)
- Webcam/phone full-body mocap with face + hand tracking, AI cleanup in ms, up to 8 simultaneous subjects.
- Subscription: Free trial; Starter $12/mo, Professional $48/mo, Studio $96/mo.
- Relevance: **8** — webcam-to-Unity-animation pipeline is finally production-grade for solo devs without mocap suits.
- RTX Windows: **yes** (cloud processing — local GPU not required).
- https://www.strayspark.studio/blog/ai-mocap-indie-developers-2026-comparison

### Concept art / 2D asset generation

**Scenario** (ongoing rollout of style-consistent batch + tiling)
- Trains on your art bible and outputs on-style 2D batches, with tiling-aware generation for game textures.
- Subscription: Free trial; Creator $39/mo, Pro $99/mo, Team $299/mo.
- Relevance: **7** — only AI 2D tool that respects a custom style well enough to be usable on a single-artist project; tiling logic added in 2026.
- RTX Windows: **yes** (hosted).
- https://www.scenario.com/

### AI-assisted coding (beyond Copilot)

**Coplay Unity MCP v9.7.0 / v9.7.1** (released 2026-05-22 / 2026-05-24) — ★Upgrade (EAI, indirectly)
- MCP bridge giving Claude / Codex / Cursor / Copilot direct hands on the Unity Editor: manage assets, edit scripts, control scenes, run play mode.
- Free (open source, MIT). Coplay also sells a hosted Aura assistant on top.
- Relevance: **10** — this is *the* answer to "AI coding beyond Copilot" for Unity. Lets your LLM iterate on enemy AI scripts and immediately test in editor.
- RTX Windows: **yes**.
- https://github.com/CoplayDev/unity-mcp/releases

**Unity AI (Beta 2026)**
- First-party in-editor assistant: GPT/Llama backed, generates code + assets + playable scenes inside Unity, project-context aware.
- Free during beta; pricing post-GA unannounced.
- Relevance: **8** — official path with editor context that third parties can't match; still beta, expect rough edges.
- RTX Windows: **yes**.
- https://unity.com/features/ai

### Procedural generation / AI level design

No notable solo-friendly releases this window. Promethean AI rolled out free Asset Management in 2026 but that's tangential. MapMagic 2 and Dungeon Architect remain the practical Unity-side choices.

### Voice acting / audio generation

**ElevenLabs SB1 Infinite Soundboard** (released 2026-05-27)
- Text-to-SFX engine wrapped in a soundboard UI; describe any sound, get it back; clips up to 30s with loop param for ambient.
- Subscription: Free 10k credits/mo with attribution; Starter $5/mo, Creator $22/mo, Pro $99/mo. SFX usage burns credits.
- Relevance: **9** — replaces hunting freesound for placeholder SFX; covers door creaks, footsteps, ambience, monster vocals.
- RTX Windows: **yes** (hosted).
- https://elevenlabs.io/blog/how-we-created-a-soundboard-using-elevenlabs-sfx-api

**TTS-Audio-Suite v4.22.7 for ComfyUI** — ★Upgrade (CUI)
- 11 voice engines (Echo-TTS, Qwen3-TTS with voice design, CosyVoice3) inside ComfyUI nodes.
- Free (open source).
- Relevance: **7** — if you already live in ComfyUI, this turns it into a voice pipeline; otherwise overkill vs ElevenLabs.
- RTX Windows: **yes** (12GB+ for the heavier engines).
- https://www.blog.brightcoding.dev/2026/06/05/tts-audio-suite-the-revolutionary-voice-toolkit-for-comfyui

## Non-AI but Powerful

### Unity Asset Store

**Utility Intelligence (GO) v3 — 3.1.5** (updated 2026-05-20) — ★Upgrade (EAI)
- Infinite Axis Utility System + FSM + behavior tree + planner + GOAP in one framework, GameObject-based.
- €115.01 one-time (Asset Store extension asset).
- Relevance: **9** — direct upgrade to hand-rolled FSM or basic behavior trees; the IAUS + planner combo is rare at this price.
- RTX Windows: **yes** (Unity package, no GPU dependency).
- https://assetstore.unity.com/packages/tools/behavior-ai/utility-intelligence-go-v3-utility-ai-framework-308338

**Utility Intelligence (ECS)** (updated 2026-04-19) — ★Upgrade (EAI)
- Same framework adapted for DOTS / ECS workflows.
- Paid (Asset Store, similar price band).
- Relevance: **6** — only worth it if you're already on DOTS; otherwise stick to GO v3.
- RTX Windows: **yes**.
- https://assetstore.unity.com/packages/tools/behavior-ai/utility-intelligence-ecs-utility-ai-framework-for-dots-316295

**May 2026 Free Assets drop**
- Gridbox Lite, FlexibleGlassDestructor, Easy Censor URP, Low Poly Interior Pack, Modular Racing Circuit Lowpoly Free.
- Free.
- Relevance: **5** — grab whatever fits your prototype.
- RTX Windows: **yes**.
- https://assetstore.unity.com/lists/free-assets-may-2026-6872837091333

### Blender addons

**Poliigon Blender Addon** (updated 2026-05-28) — ★Upgrade (B→U adjacent)
- In-Blender browser/import for 6000+ models, materials, HDRIs.
- Subscription: Free tier (limited downloads); Starter $14.95/mo, Personal $24.95/mo, Studio $99/mo.
- Relevance: **6** — texture/material library is the speed-up here; B→U is downstream.
- RTX Windows: **yes**.
- https://www.poliigon.com/blender

**BlendUnity Ultimate Toolkit** (released recently) — ★Upgrade (B→U)
- Complete two-sided Blender→Unity pipeline: fixes scale, materials, colliders, and re-syncs without re-doing the Unity-side fixes on every mesh edit. Includes Unity companion.
- Paid (Blender Market / Gumroad, one-time).
- Relevance: **9** — explicitly targets every recurring B→U friction point. If you do this round-trip more than once a week, it pays for itself fast.
- RTX Windows: **yes**.
- https://blenderartists.org/t/blendunity-ultimate-toolkit-complete-blender-unity-pipeline-released/1635767

**Export Helper Pro** (Michael Bridges, rebuilt for Blender 5) — ★Upgrade (B→U)
- Lightweight FBX/glTF/OBJ export presets for Unity/Unreal/Godot.
- Free (Gumroad, public testing).
- Relevance: **8** — free alternative to BlendUnity if you only need clean exports without the round-trip syncing.
- RTX Windows: **yes**.
- https://digitalproduction.com/2026/02/06/one-click-exports-and-zero-patience-for-menus-from-blender-to-unreal-unity-and-more/

**Unity Jobs Profiler (experimental package)**
- Dedicated profiler view for the Job System: scheduling, dependencies, worker stalls.
- Free (Unity package).
- Relevance: **7** — only if you've written Burst/Jobs code; if you haven't, ignore.
- RTX Windows: **yes**.
- https://discussions.unity.com/t/introducing-the-jobs-profiler-package-experimental-release/1691249

**Embrace Unity SDK enhancements (2026)**
- Auto-instrumentation for telemetry, lower-friction integration.
- Free tier for indies; paid tiers for production volume.
- Relevance: **5** — overkill solo unless shipping live-ops.
- RTX Windows: **yes**.
- https://embrace.io/blog/best-unity-monitoring-tools/

### Pipeline automation

**Arielle (Gamers Home)** — AI Producer Co-Pilot
- Ingests a GDD and outputs a multi-discipline production roadmap with auto-linked dependencies.
- Pricing: tiered subscription (check site, free trial available).
- Relevance: **4** — solo dev rarely needs cross-discipline scheduling; useful for grant/pitch deck planning.
- RTX Windows: **yes** (hosted).
- https://gamershome.gg/blog/7-best-project-management-tools-for-game-development-in-2026-ranked-reviewed

### Playtesting / analytics

**Pretty Good Playtesting**
- Bundles OBS with your build, uses AI to surface highlight/bug moments in the recordings.
- Pricing: free tier; paid plans for cloud storage and team features.
- Relevance: **8** — biggest pain in solo playtesting is watching 30 hours of footage. This is the fix.
- RTX Windows: **yes**.
- https://pgpt.dev/

**Playcocola**
- Indie-focused playtesting with synced gameplay + voice + comment streams.
- Pricing: free for testers; paid tiers for devs (check site).
- Relevance: **7** — fast feedback loop, lower friction than recruiting your own testers.
- RTX Windows: **yes**.
- https://playcocola.com/

**GameAnalytics**
- Free analytics SDK for Unity; the long-time default for indie funnels.
- Free at entry level.
- Relevance: **7** — not new but worth flagging if you haven't added analytics yet.
- RTX Windows: **yes**.
- https://gameanalytics.com/
